Name

Synopsis

ericmrak [OPTION]... [PROJECT]...

Description

Eric Mrak is a software developer in the Bay Area. He specializes in cloud infrastructure development and has degrees in Computer Science and Human-Computer Interaction. He is currently employed at Brex as a Senior Software Engineer on the Cloud Infrastructure team.

Options

-l, --language=LANG
specify a language for Mrak to use
-t, --tool=TOOL
specify a build tool for Mrak to use
-f, --platform=PLATFORM
specify a cloud platform for Mrak to use
-d, --databases=DB
specify a database for Mrak to use
-v, --verbose
increase Mrak's verbal feedback
-h, --help
ask Mrak for help
--version
show version information
