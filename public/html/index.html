ERICMRAK(1) Software Developers ERICMRAK(1)

Name

Eric "Mrak" Mrak - distributed cloud computing platform engineer

Synopsis

ericmrak [OPTION]... [PROJECT]...

Description

Eric Mrak is a software developer in the Bay Area. He specializes in cloud infrastructure development and has degrees in Computer Science and Human-Computer Interaction. He is currently employed at Fastly as a Senior Site Reliability Engineer.

Options

-l, --language=LANG
specify a language for Mrak to use
can be specified more than once
-t, --tool=TOOL
specify a build tool for Mrak to use
can be specified more than once
-p, --platform=PLATFORM
specify a cloud platform for Mrak to use
can be specified more than once
-d, --database=DB
specify a database for Mrak to use
can be specified more than once
-v, --verbose
increase Mrak's verbal feedback
can be specified more than once
-h, --help
ask Mrak for help
--version
show version information

Examples

ericmrak OPERATION_K8S \
  --platform=AWS       \
  --tool=EKS           \
  --tool=terraform     \
  --language=golang    \
  --database=Aurora
